## 📌 Loan Calculator

Calculates monthly payments, total interest, and repayment duration based on loan amount, interest rate, and tenure.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Computes monthly payments based on loan amount, interest rate, and duration.
* Displays total interest and overall repayment amount clearly.
* Focused on accurate calculations with a clean user interface.
